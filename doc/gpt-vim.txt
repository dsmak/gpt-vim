INTRODUCTION                                                  *gptvim-intro*

The GPT-Vim plugin provides an intuitive and user-friendly interface for Vim
users to interact with the GPT (Generative Pre-trained Transformer) tool
within their text editor. This plugin allows users to generate conversational
responses with ease, as well as manage their saved chat sessions.

The documentation is generated by the GPT-Vim plugin.

INSTALLATION                                           *gptvim-installation*

In order to install GPT-Vim, perform the following steps:

* Install the 'tiktoken', 'openai', and 'sqlite3' Python modules by running:

    ```vim
    pip install tiktoken openai sqlite3
    ```

* Install the GPT-Vim plugin using your preferred plugin manager. For example,
  using vim-plug:

    ```vim
    Plug 'kamou/gpt-vim'
    ```

* Set the variable `g:gpt#plugin_dir` to specify the directory that you would
  like to store your saved chat sessions in.

    ```vim
    let g:gpt#plugin_dir = "/path/to/your/directory"
    ```

USAGE                                                         *gptvim-usage*

GPT-Vim provides two widgets: the `chat` widget and the `conversation` widget.
The `chat` widget is used for generating conversational responses with the GPT
tool, while the `conversation` widget is used for managing saved chat sessions.

##### Launching the Chat Widget                         *gptvim-launch-chat*

                                                                *gptvim-gpa*
gpa                      Lauch the Chat widget.


You can change these mappings like this:

```vim
nmap <silent> gpa <Plug>(gpt-vim-assist)
xmap <silent> gpa <Plug>(gpt-vim-assist-vis)
vmap <silent> gpa <Plug>(gpt-vim-assist-vis)
```
In visual mode, the selection will be appended to the prompt.

##### Chat Mappings                             *gptvim-chatwidget-mappings*

- `q`: Hide the chat widget.
- `r`: Reset the chat widget to its default state.
- `s`: Save the current chat conversation to the list of saved sessions.
- `L`: List the saved chat session summaries in the conversation widget.
- `c`: Cancel the current stream. The current answer will be lost.

##### Conversation Mappings             *gptvim-conversationwidget-mappings*

- `<CR>`: Load the conversation under the cursor.
- `d`: Delete the selected chat session.
- `q`: Hide the conversation widget.
- `s`: Open the conversation under the cursor in a horizontal split.
- `v`: Open the conversation under the cursor in a vertical split.
